१. भाषाका दुई स्वभाव

रूपविज्ञान (Morphology) का दुई प्रवृत्ति:

संश्लेषणात्मक

एउटै शब्दमा प्रत्यय टाँसिने।

संस्कृत: - "पिपृच्छामि" (मलाई सोध्ने इच्छा छ)

विश्लेषणात्मक

अर्थ छुट्टाछुट्टी शब्दबाट आउने।

अंग्रेजी:

नेपाली कहाँ पर्छ?

नेपाली भाषा संस्कृत जति Synthetic छैन, तर अंग्रेजी जति Analytic पनि छैन।

यो बीचमा छ! यहाँ 'परसर्ग' (Postpositions) टाँसिने आफ्नै मौलिक प्रवृत्ति छ।

२. व्याकरणीकरण

Grammaticalization

स्वतन्त्र शब्दहरू खिइँदै 'व्याकरणिक चिन्ह' बन्ने प्रक्रिया।

ऐतिहासिक यात्रा

मध्य → मज्झ → माझ → मा
वर्त्मन् → वट्ट → बाटो / बाट
दृश् → दृष्टि → देखिअ → देखि
लग्न → लागि → ले / लाई
कृत → केरअ → केरा → को
सहित → सहिअ → सित / सँग
तीर → तिर / तर्फ
सम → सम्म

"यी त पहिला छुट्टै शब्द थिए, त्यसैले छुट्टै लेख्नुपर्छ!"

यो तर्क अधुरो छ।

किनभने इतिहासले आजको Syntax तय गर्दैन।

३. रूपग्राम (Morphemes)

स्वतन्त्र (Free) र बद्ध (Bound)

स्वतन्त्र (Free)

जो एक्लै उभिन सक्छन्।
(घर, मान्छे, राम्रो)

बद्ध (Bound)

जो अर्थ दिन अर्कोमा टाँसिनै पर्छ।
(-हरू, -पन, -ले, -लाई)

जब "मा", "को", "ले" नेपालीले आत्मसात गर्‍यो, तिनीहरू बद्ध रूपग्राम भइसके।

इतिहासले हिज्जे छुट्टाउँदैन, वर्तमानको जीवित प्रयोगले निर्धारण गर्छ।

४. Morpheme Isolation Test

शब्द छुट्टाउने कि जोड्ने? Universal Test:

  1. एक्लै वाक्यमा आउन सक्छ? → छुट्टै लेख्ने
  2. एक्लै आउँदैन? → जोडेर लेख्ने

"भित्र जा"

✓ स्वतन्त्र अर्थ छ

"मा जा" / "लाई बोला"

✗ अर्थविहीन / अधुरो

तसर्थ: "घरमा" जोडेरै लेख्नुपर्छ।

६. हिन्दीको प्रभाव

  • नेपाली: "जान्छु" (One word) vs हिन्दी: "जाता हुँ" (Two words)
  • नेपाली "न" (Negation) is extremely precise & sensitive.
  • नेपाली is more Agglutinative (गर्दिन, भन्दिन).
  • Hindi has stronger Perso-Arabic influence on spacing.

अन्तिम Verdict

किन जोडेरै लेख्ने?

  1. Morpheme Test Fail: यी बद्ध रूपग्राम हुन्।
  2. Evolutionary Fact: नेपालीले यिनलाई अंग बनाइसक्यो।
  3. Cognitive Efficiency: Reading speed बढ्छ।
  4. Structural Difference: हिन्दी र नेपाली फरक छन्।
यो लहड होइन, भाषाको विज्ञान हो।
